IntensiCode >>> Projects >>> Galaxina

Description

Simple non-scrolling vertical shooter. Largely inspired by Galaga. Adding extras and more level variations. Graphics stolen from LOST GARDEN and Galaxina - The Movie.

This is work in progress! Not even an alpha release!

    J2SE build for running Galaxina via Java Web Start. Requires Java Runtime Environment 1.4 or later. Uses the RunME J2ME emulation layer. Includes full-screen support (Alt-Enter).

  • Galaxina :: Run Galaxina from your Desktop

    J2SE build for running Galaxina from your Desktop. Requires Java Runtime Environment 1.4 or later. Uses the RunME J2ME emulation layer. Includes full-screen support (Alt-Enter).
